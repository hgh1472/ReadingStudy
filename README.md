## Effective Java 3
>Effective Java 3를 읽고 issue를 통해 공부하는 스터디입니다. <br>

- **일주일을 기준**으로 다음 주제로 넘어갑니다.
    - 스터디원들은 번갈아가며 `Leader`를 담당합니다.
    - EX) 월~일 기준으로 진행
      
    - **월 ~ 금** : 스터디원들은 자유롭게 git issue를 통해 질문합니다. 질문뿐만 아니라, 남겨두고 싶은 예시나 자유롭게 issue를 통해 기록합니다.
      
    - **토요일** : 스터디원이 모여 **주제에 대해 토의하는 시간**입니다. 자유롭게 주제에 대해 이야기하고 추가적인 질문 및 레퍼런스 공유 등 다양하게 나눕니다. **모두가 최대한 책 내용을 이해하고 참여**합니다.
    - **일요일** : 해당 주차의 Leader는 그 주의 **주제에 대한 책 내용, issue, 토의 내용을 정리**하여 **노션과 git에 업로드**합니다.
      
- 질문 : 질문은 주로 Git issue를 통해 질문합니다.
    - 처음 보는 용어에 대한 이해
      
    - 글로만 적혀있는 기술에 대한 사용법 질문
    - 예제 코드에 대한 설명
    - 문맥에 대한 이해
    - 코드에 대한 질문
    - 관련 기술 사용 예시 등
- 답변 : issue에 대해서는 자유롭게 해결합니다.
    - 구글링을 이용한 기술 블로그 참조
      
    - 공식 자바 문서
    - 실제로 코드 작성하여 해결
    - 이외 다양한 방법
